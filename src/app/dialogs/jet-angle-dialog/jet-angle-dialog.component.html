These jet cone half angles were measured from a small sample of neutron stars. The measurements were fitted to a smooth S-shaped curve (a sigmoid) and then adjusted with small quadratic corrections to improve accuracy for extreme stars.

Model constants

  • Amin = -83.8389
  • Amax = -60.8896
  • k = 2.2037
  • x0 = -5.0497
  • alpha = 0.001517
  • gamma_sr = 0.724671
  • gamma_rot = -0.025587
  • gamma_age = 0.045594

How the prediction is built

1) Combine the inputs into a single predictor:

x = ln(solarRadius / sqrt(rotationalPeriod)) + 0.001517 × ln(age)

2) Pass the predictor through the sigmoid (S-shaped) curve:

Angle_sigmoid = Amin + (Amax - Amin) / (1 + exp(-k * (x - x0)))

3) Add small quadratic corrections to account for outliers:

corrections = gamma_sr * ln(solarRadius)^2 + gamma_rot * ln(rotationalPeriod)^2 + gamma_age * ln(age)^2

4) Final prediction: Angle_pred = Angle_sigmoid + corrections

Note: the displayed model is a simplified regression of a small sample. The uncertainty is large — the tooltip shows a conservative ±5° estimate.
